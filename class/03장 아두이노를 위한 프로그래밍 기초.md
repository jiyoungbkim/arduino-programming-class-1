# 아두이노를 위한 프로그래밍 기초

## 정보를 담는 그릇 : 변수

### 변수의 종류

| 변수타입 | 저장 가능 값                                   |
| -------- | ---------------------------------------------- |
| int      | 정수형 데이터 (-32,768~32,767)                 |
| long     | 정수형 데이터 (-2,147,483,648~2,147,483,647)   |
| byte     | 부호 없는 숫자 (0~255)                         |
| float    | 실수형 데이터 (-3.4028235E+38 ~ 3.4028235E+38) |
| char     | 하나의 문자를 저장                             |
| string   | 문자열을 저장                                  |
| array[]  | 배열 값 저장                                   |
| const    | 상수(변하지 않는 고정된 값) 선언               |
